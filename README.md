# xp_CAPTCHA_api V2.2

## 注意
* 默认使用jdk1.8编译
* 在最新版的burp2.x中jdk为1x,会导致插件不可用,使用jdk8编译到不行，请下载jdk16版本试试,若还不行，请自行下载源码使用当前电脑的jdk1x进行编译,谢谢。
* 爆破时，记得把线程设置为1。

*********

### 说明
xp_CAPTCHA_api
* 验证码识别，支持计算类验证码
* burp插件
* 支持验证码返回包为json格式

**********
