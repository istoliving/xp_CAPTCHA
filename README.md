# xp_CAPTCHA_api V2.1

## 注意
* 默认使用jdk1.8编译
* 在最新版的burp2.x中jdk为1x,会导致插件不可用,使用jdk8编译到不行，请下载jdk16版本试试,若还不行，请自行下载源码使用当前电脑的jdk1x进行编译,谢谢。
* 爆破时，记得把线程设置为1。

请先http://www.kuaishibie.cn 处注册账号。（由于调用这个平台的接口，请先注册，收费是1块钱识别500次）
